题目编号：191

题目：位1的个数

难度：简单

考察范围：位运算

题干：给定一个无符号整数，返回其二进制表达式中数字位数为 '1' 的个数（也被称为汉明重量）。

解题思路：使用位运算，每次将 n 的最后一位与 1 进行与运算，如果结果为 1，则计数器加 1，然后将 n 右移一位，继续进行与运算，直到 n 变为 0。

解决方案：

```cpp
class Solution {
public:
    int hammingWeight(uint32_t n) {
        int count = 0;
        while (n != 0) {
            if (n & 1) {
                count++;
            }
            n >>= 1;
        }
        return count;
    }
};
```

算法复杂度：时间复杂度为 O(logn)，空间复杂度为 O(1)。
