题目编号：724

题目：寻找数组的中心索引

难度：简单

考察范围：数组、前缀和

题干：给定一个整数类型的数组 nums，请编写一个能够返回数组“中心索引”的方法。我们是这样定义数组中心索引的：数组中心索引的左侧所有元素相加的和等于右侧所有元素相加的和。如果数组不存在中心索引，那么我们应该返回 -1。如果数组有多个中心索引，那么我们应该返回最靠近左边的那一个。

解题思路：使用前缀和，先计算出数组的总和，然后从左往右遍历数组，每次计算当前位置左侧的元素和，如果等于总和减去当前位置和左侧元素和再减去当前位置的值，那么当前位置就是中心索引。

解决方案：

```kotlin
fun pivotIndex(nums: IntArray): Int {
    val sum = nums.sum()
    var leftSum = 0
    for (i in nums.indices) {
        if (leftSum == sum - leftSum - nums[i]) {
            return i
        }
        leftSum += nums[i]
    }
    return -1
}
```

算法复杂度：时间复杂度为 O(n)，空间复杂度为 O(1)。
