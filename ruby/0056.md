题目编号：56

题目：合并区间

难度：中等

考察范围：数组、排序、贪心算法

题干：给出一个区间的集合，请合并所有重叠的区间。

示例 1:

输入: [[1,3],[2,6],[8,10],[15,18]]
输出: [[1,6],[8,10],[15,18]]
解释: 区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].

示例 2:

输入: [[1,4],[4,5]]
输出: [[1,5]]
解释: 区间 [1,4] 和 [4,5] 可被视为重叠区间。

解题思路：首先按照区间的左端点进行排序，然后遍历区间集合，如果当前区间的左端点小于等于上一个区间的右端点，则说明两个区间有重叠，将它们合并，否则将当前区间加入结果集。

解决方案：

```ruby
def merge(intervals)
  return [] if intervals.empty?
  intervals.sort! {|a, b| a[0] <=> b[0]}
  res = [intervals[0]]
  intervals[1..-1].each do |interval|
    if interval[0] <= res[-1][1]
      res[-1][1] = [res[-1][1], interval[1]].max
    else
      res << interval
    end
  end
  res
end
```

算法复杂度：时间复杂度为 O(nlogn)，空间复杂度为 O(n)。
