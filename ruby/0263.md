题目编号：263

题目：丑数

难度：简单

考察范围：数学、动态规划

题干：编写一个程序判断给定的数是否为丑数。丑数就是只包含质因数 2, 3, 5 的正整数。

解题思路：如果一个数是丑数，那么它一定可以被 2、3、5 整除，因此我们可以不断地将这个数除以 2、3、5，直到无法再被整除为止。最后判断剩余的数是否为 1，如果是 1 则说明原数是丑数，否则不是丑数。

解决方案：

```ruby
def is_ugly(num)
  return false if num <= 0
  while num % 2 == 0
    num /= 2
  end
  while num % 3 == 0
    num /= 3
  end
  while num % 5 == 0
    num /= 5
  end
  num == 1
end
```

算法复杂度：时间复杂度为 O(log n)，空间复杂度为 O(1)。
