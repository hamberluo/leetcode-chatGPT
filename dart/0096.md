题目编号：96

题目：不同的二叉搜索树

难度：中等

考察的知识点：动态规划

题干：

给定一个整数 n，求以 1 ... n 为节点组成的二叉搜索树有多少种？

典型解题思路：

1. 首先，我们可以将问题转化为子问题，即求以 1 ... i 为节点组成的二叉搜索树有多少种，其中 i <= n。

2. 对于每个 i，我们可以将其作为根节点，左子树由 1 ... i-1 组成，右子树由 i+1 ... n 组成。

3. 对于左子树和右子树，我们可以递归地求解它们的二叉搜索树数量。

4. 最后，将左子树和右子树的数量相乘，即可得到以 i 为根节点的二叉搜索树数量。

5. 将所有以 1 ... n 为节点组成的二叉搜索树数量相加，即可得到最终的答案。

解决方案（用 dart 语言）：

```dart
int numTrees(int n) {
  List<int> dp = List.filled(n + 1, 0);
  dp[0] = 1;
  dp[1] = 1;
  for (int i = 2; i <= n; i++) {
    for (int j = 1; j <= i; j++) {
      dp[i] += dp[j - 1] * dp[i - j];
    }
  }
  return dp[n];
}
```

算法复杂度：时间复杂度为 O(n^2)，空间复杂度为 O(n)。
