题目编号：481

题目：神奇字符串

难度：中等

考察范围：字符串、模拟

题干：神奇的字符串 S 只包含 '1' 和 '2'，并遵守以下规则：

字符串 S 是神奇的，因为串联字符 '1' 和 '2' 的连续出现次数会生成字符串 S 本身。

字符串 S 的前几个元素如下：S = “1221121221221121122 ......”

如果我们将 S 中连续的 1 和 2 进行分组，它将变成：

1 22 11 2 1 22 1 22 11 2 11 22 ......

并且每个组中 '1' 或 '2' 的出现次数分别是：

1 2 2 1 1 2 1 2 2 1 2 2 ......

你可以看到上面的出现次数就是 S 本身。

给定一个整数 N 作为输入，返回神奇字符串 S 中前 N 个数字中的 '1' 的数目。

注意：N 不会超过 100,000。

解题思路：模拟

我们可以先生成一个初始的字符串 "122"，然后从第三个字符开始，每次根据前一个字符的值来生成下一个字符，直到生成长度为 N 的字符串。在生成的过程中，我们可以记录下每个字符 '1' 的出现次数，最后返回前 N 个字符中 '1' 的数量即可。

解决方案：

```rust
impl Solution {
    pub fn magical_string(n: i32) -> i32 {
        if n == 0 {
            return 0;
        }
        if n <= 3 {
            return 1;
        }
        let mut s = vec![0; n as usize];
        s[0] = 1;
        s[1] = 2;
        s[2] = 2;
        let mut head = 2;
        let mut tail = 3;
        let mut num = 1;
        let mut result = 1;
        while tail < n as usize {
            for _ in 0..s[head] {
                s[tail] = num;
                if num == 1 && tail < n as usize {
                    result += 1;
                }
                tail += 1;
            }
            num = 3 - num;
            head += 1;
        }
        result
    }
}
```

算法复杂度：时间复杂度为 O(N)，空间复杂度为 O(N)。
