题目编号：794

题目：有效的井字游戏

难度：简单

考察范围：字符串、逻辑判断

题干：给定一个字符串数组，代表井字游戏板，返回此次游戏是有效的还是无效的。当且仅当在以下情况下，此游戏有效：

1.板上有三个相同的符号（即'A'或'B'）形成的行，列或对角线。
2.游戏板被填满，但是没有形成三个相同的符号的行，列或对角线。

解题思路：遍历整个井字游戏板，判断是否存在三个相同的符号形成的行、列或对角线。如果存在，则返回有效，否则判断是否已经填满，如果已经填满则返回有效，否则返回无效。

解决方案：

```solidity
pragma solidity ^0.8.0;

contract TicTacToe {
    function validTicTacToe(string[] memory board) public pure returns (bool) {
        uint8 countX = 0;
        uint8 countO = 0;
        bool xWin = false;
        bool oWin = false;
        // 判断行
        for (uint8 i = 0; i < 3; i++) {
            if (board[i][0] == board[i][1] && board[i][1] == board[i][2]) {
                if (board[i][0] == 'X') {
                    xWin = true;
                } else if (board[i][0] == 'O') {
                    oWin = true;
                }
            }
        }
        // 判断列
        for (uint8 j = 0; j < 3; j++) {
            if (board[0][j] == board[1][j] && board[1][j] == board[2][j]) {
                if (board[0][j] == 'X') {
                    xWin = true;
                } else if (board[0][j] == 'O') {
                    oWin = true;
                }
            }
        }
        // 判断对角线
        if (board[0][0] == board[1][1] && board[1][1] == board[2][2]) {
            if (board[0][0] == 'X') {
                xWin = true;
            } else if (board[0][0] == 'O') {
                oWin = true;
            }
        }
        if (board[0][2] == board[1][1] && board[1][1] == board[2][0]) {
            if (board[0][2] == 'X') {
                xWin = true;
            } else if (board[0][2] == 'O') {
                oWin = true;
            }
        }
        // 统计 X 和 O 的数量
        for (uint8 i = 0; i < 3; i++) {
            for (uint8 j = 0; j < 3; j++) {
                if (board[i][j] == 'X') {
                    countX++;
                } else if (board[i][j] == 'O') {
                    countO++;
                }
            }
        }
        // 判断是否有效
        if (xWin && oWin) {
            return false;
        } else if (xWin && countX == countO) {
            return false;
        } else if (oWin && countX > countO) {
            return false;
        } else if (countX - countO > 1) {
            return false;
        } else if (!xWin && !oWin && countX + countO < 9) {
            return true;
        } else if (!xWin && !oWin && countX + countO == 9) {
            return true;
        } else if (xWin || oWin) {
            return true;
        }
        return false;
    }
}
```

算法复杂度：时间复杂度为 O(1)，空间复杂度为 O(1)。
