题目编号：640

题目：求解方程

难度：中等

考察范围：数学、字符串处理

题干：给定一个方程，求解方程，方程中包含字母和数字，字母表示未知数，数字表示系数，方程中只包含加号或减号，没有乘号，例如 "x+5-3+x=6+x-2"。

解题思路：首先将方程左右两边分别拆分成未知数和常数两个部分，然后将未知数和常数分别相加，得到方程的标准形式，即 ax + b = cx + d。然后将未知数移到左边，常数移到右边，得到方程的一般形式，即 ax = b。最后求解未知数 x 的值即可。

解决方案：

```solidity
pragma solidity ^0.8.0;

contract EquationSolver {
    function solveEquation(string memory equation) public pure returns (int256) {
        int256 a = 0;
        int256 b = 0;
        int256 c = 0;
        int256 d = 0;
        int256 coef = 1;
        bool isLeft = true;
        bool isNegative = false;
        bytes memory eqBytes = bytes(equation);
        for (uint i = 0; i < eqBytes.length; i++) {
            if (eqBytes[i] == '+') {
                if (isNegative) {
                    coef = -1;
                } else {
                    coef = 1;
                }
                if (isLeft) {
                    b += coef * c;
                    c = 0;
                } else {
                    d += coef * c;
                    c = 0;
                }
                isNegative = false;
            } else if (eqBytes[i] == '-') {
                if (isNegative) {
                    coef = 1;
                } else {
                    coef = -1;
                }
                if (isLeft) {
                    b += coef * c;
                    c = 0;
                } else {
                    d += coef * c;
                    c = 0;
                }
                isNegative = true;
            } else if (eqBytes[i] == '=') {
                if (isNegative) {
                    coef = 1;
                } else {
                    coef = -1;
                }
                if (isLeft) {
                    b += coef * c;
                    c = 0;
                } else {
                    d += coef * c;
                    c = 0;
                }
                isNegative = false;
                isLeft = false;
            } else if (eqBytes[i] >= '0' && eqBytes[i] <= '9') {
                c = c * 10 + int256(eqBytes[i] - '0');
            } else if (eqBytes[i] >= 'a' && eqBytes[i] <= 'z') {
                if (isNegative) {
                    coef = -1;
                } else {
                    coef = 1;
                }
                if (isLeft) {
                    a += coef * (c == 0 ? 1 : c);
                } else {
                    a -= coef * (c == 0 ? 1 : c);
                }
                c = 0;
                isNegative = false;
            }
        }
        if (isNegative) {
            coef = -1;
        } else {
            coef = 1;
        }
        if (isLeft) {
            b += coef * c;
        } else {
            d += coef * c;
        }
        return b == d ? 0 : (b - d) / a;
    }
}
```

算法复杂度：时间复杂度为 O(n)，空间复杂度为 O(1)。
