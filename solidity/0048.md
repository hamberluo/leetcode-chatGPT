题目：旋转图像

难度：中等

考察范围：数组、矩阵操作

题干：

给定一个 n × n 的二维矩阵表示一个图像。

将图像顺时针旋转 90 度。

说明：

你必须在原地旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要使用另一个矩阵来旋转图像。

示例 1:

给定 matrix = 
[
  [1,2,3],
  [4,5,6],
  [7,8,9]
],

原地旋转输入矩阵，使其变为:
[
  [7,4,1],
  [8,5,2],
  [9,6,3]
]

示例 2:

给定 matrix =
[
  [ 5, 1, 9,11],
  [ 2, 4, 8,10],
  [13, 3, 6, 7],
  [15,14,12,16]
], 

原地旋转输入矩阵，使其变为:
[
  [15,13, 2, 5],
  [14, 3, 4, 1],
  [12, 6, 8, 9],
  [16, 7,10,11]
]

解题思路：

这道题目可以通过矩阵操作来解决。我们可以先将矩阵沿着主对角线翻转，然后再将每一行沿着中心翻转即可。

例如，对于示例 1，我们可以先将矩阵沿着主对角线翻转，得到：

[
  [1,4,7],
  [2,5,8],
  [3,6,9]
]

然后再将每一行沿着中心翻转，得到：

[
  [7,4,1],
  [8,5,2],
  [9,6,3]
]

解决方案：

算法复杂度：时间复杂度为 O(n^2)，空间复杂度为 O(1)。

```solidity
pragma solidity ^0.8.0;

contract Solution {
    function rotate(uint[][] memory matrix) public {
        uint n = matrix.length;
        // 沿着主对角线翻转
        for (uint i = 0; i < n; i++) {
            for (uint j = i + 1; j < n; j++) {
                uint temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }
        // 每一行沿着中心翻转
        for (uint i = 0; i < n; i++) {
            for (uint j = 0; j < n / 2; j++) {
                uint temp = matrix[i][j];
                matrix[i][j] = matrix[i][n - j - 1];
                matrix[i][n - j - 1] = temp;
            }
        }
    }
}
```
