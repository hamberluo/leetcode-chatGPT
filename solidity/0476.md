题目编号：476

题目：数字的补数

难度：简单

考察范围：位运算

题干：给定一个正整数，输出它的补数。补数是对该数的二进制表示取反。

解题思路：将原数转换为二进制，再将每一位取反，最后将结果转换为十进制即可。

解决方案：

```solidity
pragma solidity ^0.8.0;

contract Solution {
    function findComplement(uint num) public pure returns (uint) {
        uint temp = num;
        uint bit = 1;
        while (temp > 0) {
            num = num ^ bit;
            bit = bit << 1;
            temp = temp >> 1;
        }
        return num;
    }
}
```

算法复杂度：时间复杂度为 O(log n)，空间复杂度为 O(1)。
