题目编号：371

题目：两整数之和

难度：简单

考察范围：位运算

题干：给你两个整数，返回它们的和。输入的整数在 32 位有符号整数范围内。

解题思路：使用位运算实现加法。首先不考虑进位，两个数异或的结果就是不考虑进位的加法结果；然后考虑进位，两个数相与并左移一位的结果就是进位的结果。将这两个结果相加，就是最终的结果。

解决方案：

```java
class Solution {
    public int getSum(int a, int b) {
        while (b != 0) {
            int carry = (a & b) << 1;
            a = a ^ b;
            b = carry;
        }
        return a;
    }
}
```

算法复杂度：时间复杂度为 O(1)，空间复杂度为 O(1)。
