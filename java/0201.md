题目编号：201

题目：数字范围按位与

难度：中等

考察范围：位运算

题干：

给定范围 [m, n]，其中 0 <= m <= n <= 2147483647，返回此范围内所有数字的按位与（包含 m, n 两端点）。

示例 1:

输入: [5,7]
输出: 4

示例 2:

输入: [0,1]
输出: 0

解题思路：

题目要求我们求出一个区间内所有数字的按位与，我们可以先来看一下按位与的定义：

按位与（&）：两个数的二进制位，如果都为1，则该位的结果为1，否则为0。

我们可以先将区间的两个端点进行按位与操作，得到的结果就是这两个数的二进制位的公共前缀，因为如果这个公共前缀的某一位为0，那么这个区间内的所有数字在这一位上的结果都为0，因为只要有一个数在这一位上为0，按位与的结果就为0了。

所以我们可以先将m和n进行按位与操作，然后将结果向右移动直到m和n相等，这个结果就是区间内所有数字的按位与结果。

解决方案：

```java
class Solution {
    public int rangeBitwiseAnd(int m, int n) {
        int shift = 0;
        while (m < n) {
            m >>= 1;
            n >>= 1;
            shift++;
        }
        return m << shift;
    }
}
```

算法复杂度：时间复杂度为O(logn)，空间复杂度为O(1)。
