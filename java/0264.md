题目编号：264

题目：丑数 II

难度：中等

考察范围：动态规划、堆

题干：编写一个程序，找出第 n 个丑数。

丑数就是只包含质因数 2、3 和/或 5 的正整数。

示例:

输入: n = 10
输出: 12
解释: 1, 2, 3, 4, 5, 6, 8, 9, 10, 12 是前 10 个丑数。

说明:  

1 是丑数。
n 不超过1690。

解题思路：动态规划

我们可以用动态规划的思想来解决这个问题。我们可以定义一个数组 dp，其中 dp[i] 表示第 i 个丑数。那么我们如何得到 dp[i] 呢？

我们可以将 dp[i] 分为三个部分，分别是 2 的倍数、3 的倍数和 5 的倍数。那么 dp[i] 就是这三个部分中的最小值。即：

dp[i] = min(dp[p2] * 2, dp[p3] * 3, dp[p5] * 5)

其中 p2、p3、p5 分别表示当前 2、3、5 的倍数中最小的那个数的下标。我们可以通过维护这三个下标来得到 dp[i]。

解决方案：

```java
class Solution {
    public int nthUglyNumber(int n) {
        int[] dp = new int[n];
        dp[0] = 1;
        int p2 = 0, p3 = 0, p5 = 0;
        for (int i = 1; i < n; i++) {
            dp[i] = Math.min(dp[p2] * 2, Math.min(dp[p3] * 3, dp[p5] * 5));
            if (dp[i] == dp[p2] * 2) p2++;
            if (dp[i] == dp[p3] * 3) p3++;
            if (dp[i] == dp[p5] * 5) p5++;
        }
        return dp[n - 1];
    }
}
```

算法复杂度：时间复杂度为 O(n)，空间复杂度为 O(n)。
