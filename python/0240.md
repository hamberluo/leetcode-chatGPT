题目编号：240

题目：搜索二维矩阵 II

难度：中等

考察的知识点：二分查找、分治算法

题干：

编写一个高效的算法来搜索 m x n 矩阵 matrix 中的一个目标值 target。该矩阵具有以下特性：

每行的元素从左到右升序排列。
每列的元素从上到下升序排列。

示例：

现有矩阵 matrix 如下：

[
  [1,   4,  7, 11, 15],
  [2,   5,  8, 12, 19],
  [3,   6,  9, 16, 22],
  [10, 13, 14, 17, 24],
  [18, 21, 23, 26, 30]
]

给定 target = 5，返回 true。

给定 target = 20，返回 false。

典型解题思路：

1. 从矩阵的右上角开始查找，如果当前值比目标值大，则往左移动一列，如果当前值比目标值小，则往下移动一行，直到找到目标值或者越界。

解决方案（用 python 语言）：

class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if not matrix:
            return False
        m, n = len(matrix), len(matrix[0])
        i, j = 0, n - 1
        while i < m and j >= 0:
            if matrix[i][j] == target:
                return True
            elif matrix[i][j] > target:
                j -= 1
            else:
                i += 1
        return False

算法复杂度：时间复杂度为 O(m+n)，空间复杂度为 O(1)。
