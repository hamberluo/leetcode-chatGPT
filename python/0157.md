题目编号：157

题目：寻找旋转排序数组中的最小值

难度：中等

考察的知识点：二分查找

题干：假设按照升序排序的数组在预先未知的某个点上进行了旋转。例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2]。

请找出其中最小的元素。

典型解题思路：

1. 二分查找

解决方案（用 python 语言）：

class Solution:
    def findMin(self, nums: List[int]) -> int:
        left, right = 0, len(nums) - 1
        while left < right:
            mid = (left + right) // 2
            if nums[mid] > nums[right]:
                left = mid + 1
            else:
                right = mid
        return nums[left]

算法复杂度：时间复杂度为 O(log n)，空间复杂度为 O(1)。
