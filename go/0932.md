题目编号：932

题目：漂亮数组

难度：中等

考察范围：分治算法

题干：

对于一个数组 A，如果存在某个数 x，使得将 A 中小于 x 的数按升序排列，将 A 中大于等于 x 的数按升序排列，得到的新数组 B 严格递增，那么数组 A 就是漂亮数组。

给定整数 N，返回任意漂亮数组 A（保证 N 为正整数）。

解题思路：

对于一个漂亮数组 A，将其分为两部分，分别为奇数部分和偶数部分，那么可以得到以下结论：

1. 对于奇数部分，每个数乘以 2 再减去 1，得到的数组仍然是漂亮数组；
2. 对于偶数部分，每个数乘以 2，得到的数组仍然是漂亮数组；
3. 将两个漂亮数组合并，得到的数组仍然是漂亮数组。

根据以上结论，可以使用分治算法，将 N 分为两部分，分别求出漂亮数组，然后合并即可。

解决方案：

```go
func beautifulArray(N int) []int {
    if N == 1 {
        return []int{1}
    }
    odd := beautifulArray((N + 1) / 2)
    even := beautifulArray(N / 2)
    res := make([]int, N)
    for i := 0; i < len(odd); i++ {
        res[i] = odd[i]*2 - 1
    }
    for i := 0; i < len(even); i++ {
        res[len(odd)+i] = even[i]*2
    }
    return res
}
```

算法复杂度：时间复杂度为 O(NlogN)，空间复杂度为 O(N)。
