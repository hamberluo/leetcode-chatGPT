题目编号：190

题目：颠倒二进制位

难度：简单

考察范围：位运算

题干：颠倒给定的 32 位无符号整数的二进制位。

解题思路：将原数的二进制位从低位到高位逐一取出，然后将其放到新数的对应高位上即可。

解决方案：

```go
func reverseBits(num uint32) uint32 {
    var res uint32
    for i := 0; i < 32; i++ {
        res <<= 1
        res |= num & 1
        num >>= 1
    }
    return res
}
```

算法复杂度：时间复杂度为 O(1)，空间复杂度为 O(1)。
