题目编号：178

题目：分数排名

难度：中等

考察范围：排序、SQL

题干：编写一个 SQL 查询来实现分数排名。如果两个分数相同，则两个分数排名相同，排名相同的下一个分数的排名应该是相应分数排名的下一个非重复分数的排名。

例如，下面是分数表：

```go
+----+-------+
| Id | Score |
+----+-------+
| 1  | 3.50  |
| 2  | 3.65  |
| 3  | 4.00  |
| 4  | 3.85  |
| 5  | 4.00  |
| 6  | 3.65  |
+----+-------+
```

应该输出：

```go
+-------+------+
| Score | Rank |
+-------+------+
| 4.00  | 1    |
| 4.00  | 1    |
| 3.85  | 2    |
| 3.65  | 3    |
| 3.65  | 3    |
| 3.50  | 4    |
+-------+------+
```

解题思路：使用 SQL 语句进行排序和排名。

解决方案：

```go
SELECT Score, 
       (SELECT COUNT(DISTINCT Score) 
        FROM Scores 
        WHERE Score >= s.Score) AS Rank
FROM Scores s
ORDER BY Score DESC;
```

算法复杂度：时间复杂度为 O(nlogn)，空间复杂度为 O(1)。
